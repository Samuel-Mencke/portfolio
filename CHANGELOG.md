# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-15

### Added
- Initial release of the portfolio website
- Three.js 3D animated background
- GitHub API integration with automatic repository fetching
- Project detail modal with image carousel
- Smart image filtering (excludes badges, logos, shields)
- Dark/Light mode toggle
- Contact form with SMTP support
- Responsive design for all devices
- Custom cursor effects
- Smooth scroll animations with GSAP
- Project blacklist via environment variables
- Star-based sorting for projects
- Fork detection for contributed projects
- README.md content display
- Mobile-optimized navigation

### Features
- **3D Background**: Interactive particle system using Three.js
- **GitHub Integration**: Automatically syncs repositories
- **Image Gallery**: Carousel with touch/swipe support
- **Contact Form**: Email functionality with validation
- **Theme Switching**: Dark/Light/Auto modes
- **Responsive**: Works on desktop, tablet, and mobile
- **Accessible**: Keyboard navigation and screen reader support

### Security
- Environment variables protected
- Input sanitization
- CSRF protection on forms
- No exposed credentials
